# Hangman game online
## Java Spring Boot application with WebSockets, allows two players to play with each other. 

### <a href="http://fierce-hamlet-24710.herokuapp.com/">Application deployed on Heroku server</a> 


### Built with:

#### Back-end:
* Java 8
* Spring Boot 2.0.3
* Spring WebSocket
* Lombok

#### Front-end:
* JavaScript
* jQuery
* SockJS
* Stomp Websocket
* HTML
* CSS
* JSP
* Bootstrap

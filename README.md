# Hangman game online
## Java Spring Boot application with WebSockets, allows two players to play with each other. 

### <a href="http://webskey-hangman.herokuapp.com/">Application deployed on Heroku</a> 

### Built with:

#### Back-end:
* Java 8
* Spring Boot
* Spring WebSocket
* Lombok

#### Front-end:
* JavaScript
* jQuery
* SockJS
* Stomp Websocket
* HTML
* CSS
* JSP
* Bootstrap
